# Booth Data Task: Apology Order and Emotion

This repository contains an anonymous submission for the Behavioral Science and Marketing data analysis task.

The analysis examines how apology order and self-blame attribution relate to emotional responses in survey data, with a focus on emotional positivity across different apology scenarios.

---

## Research Question

## Repository Structure

- `Rmd/`  
  R Markdown files containing the main analysis and appendix.

- `scripts/`  
  Supporting data processing and analysis scripts.

- `data_raw/`  
  Raw survey data and documentation provided with the task.

- `data_processed/`  
  Intermediate datasets created during data cleaning.

- `data_clean/`  
  Final cleaned dataset used for all reported analyses.

- `output/`  
  Figures, rendered HTML files, and result tables.

---

## Reproducibility

The analysis is fully reproducible from the raw data.

To reproduce the results:
1. Open the R Markdown files in the `Rmd/` directory.
2. Run them sequentially from top to bottom.
3. All figures and tables will be generated and saved to the `output/` directory.

---

## Data Notes

All reported results are based on the cleaned dataset in `data_clean/`.

Raw and processed datasets are retained to document data-cleaning decisions and ensure transparency.
